Existem diversos tipos de operadores que podemos usar em javascript, desde os operados lógicos, operadores matemáticos, relacionais e bitwise. Vamos ver sobre alguns deles.

## Operadores matemáticos
<br>

### Adição
A adição em javascript tem como símbolo o `+` e é usado para adicionar dois valores numéricos ou concatenar strings. Veja o exemplo:

```
const num1 = 5;
const num2 = 10;
const result = num1 + num2;

console.log(result); // o resultado é 15
```
